Hello, World! This is a Simple HTML Page

This is a basic paragraph of text. HTML uses **tags** to structure content. For example, the **<p>** tag defines a paragraph, and the **<h1>** tag defines the main heading.

Key HTML Elements Used:

  • **<!DOCTYPE html>**: Declares the document type.
  • **<html>**: The root element of the page.
  • **<head>**: Contains meta-information (like the title).
  • **<body>**: Contains the visible content of the page.
